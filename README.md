# Japan Weather API for SD2 小电视

轻量级日本气象厅（JMA）数据代理，专为 SD2 小电视等资源受限设备设计。

## 特性

- 🌤️ 天气预报（支持都道府县 → 地方 → 市区町村）
- 🚨 地震速报（支持按都道府県过滤）
- 🌊 海啸预警（沿海地区级别）
- 📍 多层级地区代码查询
- ⚡ 极简 JSON 响应（几百字节）
- 🚀 支持 Cloudflare Workers 和 Node.js 部署
- 💾 内置缓存（减少 JMA 请求）

## 响应大小对比

| 接口          | 原始数据源 | 本代理 | 压缩率 |
| ------------- | ---------- | ------ | ------ |
| tenki.jp 天气 | ~174KB     | ~270B  | 99.8%  |
| JMA 天气      | ~6KB       | ~80B   | 98.7%  |
| JMA 地震      | ~222KB     | ~130B  | 99.9%  |
| JMA 海啸      | ~14KB      | ~56B   | 99.6%  |
| JMA 地区代码  | ~262KB     | ~4.5KB | 98.3%  |
| 综合数据      | ~242KB     | ~305B  | 99.9%  |
